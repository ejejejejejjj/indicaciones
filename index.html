Paso 1: Configurar un Proyecto en la Consola de Desarrolladores de Google

  1. Abre tu navegador web y ve a la Consola de Desarrolladores de Google.
  2. Inicia sesión con tu cuenta de Google si no lo has hecho.
  3. Haz clic en el botón "Seleccionar proyecto" o "Crear proyecto" en la esquina superior derecha. Si creas un nuevo proyecto, dale un nombre significativo.
  4. Una vez que tengas el proyecto seleccionado o creado, haz clic en "Habilitar APIs y servicios" para acceder a la biblioteca de APIs.

Paso 2: Habilitar la API de Google Sheets

  1. En la página de la Consola de Desarrolladores, utiliza la barra de búsqueda para encontrar y seleccionar la API de Google Sheets.
  2. Haz clic en el botón "Habilitar" para habilitar la API de Google Sheets para tu proyecto.

Paso 3: Configurar Credenciales

  1. En la página de la API de Google Sheets, haz clic en "Credenciales" en el menú de la izquierda.
  2. Haz clic en el botón "Crear credenciales" y selecciona "ID de cliente de OAuth".
  3. Selecciona "Aplicación web" como tipo de aplicación.
  4. Ingresa un nombre para la aplicación web.
  5. En la sección "URI de redireccionamiento autorizado", ingresa la URL a la que Google Sheets redirigirá después de la autorización. Esto podría ser una página de tu sitio web donde manejarás la respuesta de autorización. Asegúrate de que esta URL esté correctamente configurada en tu aplicación.
  6. Haz clic en "Crear" para crear el ID de cliente.
  7. Una vez que hayas creado el ID de cliente, podrás ver el client_id. Guárdalo en un lugar seguro.

Paso 4: Obtener la API Key

  1. En la página de la Consola de Desarrolladores, en el menú lateral izquierdo, selecciona "Credenciales".
  2. Haz clic en "Crear credenciales" y selecciona "Clave de API".
  3. Aparecerá una ventana emergente con la API Key generada. Guárdala en un lugar seguro.

Paso 5: Obtener el Spreadsheet ID

  1. Abre la hoja de Google Sheets que deseas utilizar con la API.
  2. Copia el ID de la hoja de Google Sheets desde la URL del navegador. El ID es la cadena de caracteres entre "/d/" y "/edit" en la URL. Por ejemplo, si la URL es "https://docs.google.com/spreadsheets/d/abc123/edit", el ID del spreadsheet sería "abc123".

Paso 6: Configurar la Redirect URL

  1. Asegúrate de que la Redirect URL que ingresaste en el paso 11 coincida con la que utilizarás en tu aplicación.
¡Listo! Ahora tienes el client_id, la api key, el spreadsheet id y la redirect URL necesarios para trabajar con la API de Google Sheets en tu aplicación. Asegúrate de mantener esta información segura y no compartirla públicamente.
